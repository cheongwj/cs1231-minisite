Section 1.1 — Propositional Logic

Propositions

Definition — Proposition

A proposition is a declarative sentence that is either true or false. This quality of being true (T) or false (F) is called the truth value of the proposition.

Examples

True propositions: \(1+1=2\)

False propositions: \(4+5=6\)

Non-propositions (questions, commands, exclamations, or sentences with variables):

  • "How is the lecture?" — question
  • "Stop talking!" — command
  • \(x+y>0\) — predicate (becomes proposition only when \(x,y\) are known)

Example — Connectives

Let \(p\) be "it is hot" and \(q\) be "it is sunny."

  • \(\neg p\): "It is not hot."
  • \(\neg p\wedge q\): "It is not hot but it is sunny."
  • "It is neither hot nor sunny": \(\neg p\wedge\neg q\)
Example — Combining Connectives

Let \(p\): "\(x>0\)", \(q\): "\(x<3\)", \(r\): "\(x=3\)". Then:

  • \(q\vee r\): \(x<3\) or \(x=3\), i.e., \(x\le 3\). True when \(x=3\) or \(x=2\).
  • \(p\wedge(q\vee r)\): \(x>0\) and \(x\le 3\), i.e., \(0<x\le 3\).

Conditional Propositions

Motivation — Government Promise

"Taxes will be lowered if the growth is \(\ge 3\%\)." Let \(p\): "growth \(\ge3\%\)", \(q\): "taxes lowered".

Definition — Conditional Proposition

The conditional proposition \(p\to q\) is false when \(p\) is true and \(q\) is false; true otherwise. \(p\) is the hypothesis / premise; \(q\) is the conclusion / consequence.

\(p\)\(q\)\(p\to q\)
TTT
TFF
FTT

Key: \(p\to q\) is only false when the hypothesis is true but the conclusion is false.

Definition — Contrapositive, Converse, Inverse

Given \(p\to q\):

  • Contrapositive: \(\neg q\to\neg p\)
  • Converse: \(q\to p\)
  • Inverse: \(\neg p\to\neg q\)
Theorem
(i) Equiv. to contrapositive:
\(p\to q\equiv\neg q\to\neg p\)
(ii) Converse = inverse:
\(q\to p\equiv\neg p\to\neg q\)
(iii) NOT equiv. to converse:
\(p\to q\not\equiv q\to p\)
Example — Units Digit

"If the units digit of \(n\) is 0, then \(n\) is a multiple of 2."

  • Contrapositive: If \(n\) is NOT a multiple of 2, then its units digit is NOT 0. (equivalent)
  • Converse: If \(n\) is a multiple of 2, then its units digit is 0. (NOT equivalent — e.g. \(n=12\))
  • Inverse: If the units digit is NOT 0, then \(n\) is NOT a multiple of 2. (NOT equivalent)
Definition — Equivalence

Two compound propositions \(p\) and \(q\) are equivalent, written \(p\equiv q\), if they always have the same truth value.

"If", "Only if", "Iff" — Language Reference

PhraseLogical formNotes
"if \(p\) then \(q\)"\(p\to q\)\(p\) sufficient for \(q\)
"\(p\) implies \(q\)"\(p\to q\)
"\(p\) is sufficient for \(q\)"\(p\to q\)
"\(q\) is necessary for \(p\)"\(p\to q\)\(q\) must hold for \(p\) to hold
"\(p\) only if \(q\)"\(p\to q\)NOT "\(q\to p\)" — common error!
"\(p\) if and only if \(q\)"\(p\leftrightarrow q\)\(p\) necessary AND sufficient for \(q\)
Common error — "only if"

"You can have ice cream only if you finish dinner" means: if you don't finish dinner, you can't have ice cream, i.e., ice cream \(\to\) finish dinner. It does NOT mean "if you finish dinner, you get ice cream."

Operator Precedence

Precedence (highest to lowest)

\[\neg \quad\succ\quad\wedge,\vee\quad\succ\quad\to,\leftrightarrow\]

  • \(\wedge\) and \(\vee\) have the same precedence (use parentheses to disambiguate)
  • \(\to\) and \(\leftrightarrow\) have the same precedence
  • \(\neg p\vee q\) means \((\neg p)\vee q\), NOT \(\neg(p\vee q)\)
  • Cannot write \(p\vee q\wedge r\) — must write \((p\vee q)\wedge r\) or \(p\vee(q\wedge r)\)

Translating English Sentences

Example — Internet Access

"You can access the internet from campus only if you are a computer science major or you are not a freshman."

Let \(a\): "access internet", \(c\): "CS major", \(f\): "freshman". Answer: \(a\to(c\vee\neg f)\)

Example — Diagnostic Buffer

Express simultaneously: "The diagnostic message is stored in the buffer or it is retransmitted", "The message is NOT stored in the buffer", "If stored in the buffer, then it is retransmitted."

Let \(s\): "stored in buffer", \(r\): "retransmitted". The three conditions are \(s\vee r\), \(\neg s\), \(s\to r\). All are simultaneously true when \(s=F\) and \(r=T\).

Definition — Tautology

A tautology is a compound proposition that is always true.

\(p\)\(\neg p\)\(p\vee\neg p\)

Definition — Contradiction

A contradiction is a compound proposition that is always false.

Definition — Contingency

A contingency is a compound proposition that is neither a tautology nor a contradiction. Example: \(p\wedge q\).

Section 1.3 — Predicates & Quantifiers

Definition — Predicate

A predicate is a sentence that contains a finite number of variables and becomes a proposition when the variables are specialised. The domain of a predicate variable is the set of all values that may be substituted.

Examples of Predicates
  • \(P(x)\): "\(x\) is a student at NUS", domain = residents of Clementi. \(P(\text{John Tan})\) is a proposition.
  • \(Q(x)\): "\(x^2>x\)", domain \(\mathbb{R}\). \(Q(2)=T\), \(Q(0.5)=F\).
  • \(R(x,y,z)\): "\(x+y>z\)", domain \(\mathbb{R}\). \(R(1,2,3)=F\), \(R(0,1,-1)=T\).

Universal Quantifier

Definition — Universal Quantification

The universal quantification of \(P(x)\) with domain \(D\) is \(\forall x\in D,\,P(x)\).

  • TRUE if \(P(x)\) is true for all \(x\in D\).
  • FALSE if \(P(x)\) is false for at least one \(x\in D\). Such an \(x\) is a counter example.

Other phrasings of \(\forall\): "for every", "all of", "for each", "given any", "for arbitrary".

Examples
  • \(\forall x\in\{1,2,3,4,5\},\,x^2\ge x\) is TRUE (verify: \(1\ge1,4\ge2,9\ge3,16\ge4,25\ge5\)).
  • \(\forall x\in\mathbb{R},\,x^2\ge x\) is FALSE (counter example: \(x=0.5\), \(0.25<0.5\)).
  • "Every integer is also rational": \(\forall x\in\mathbb{Z},\,x\in\mathbb{Q}\). TRUE.

Existential Quantifier

Definition — Existential Quantification

The existential quantification of \(P(x)\) with domain \(D\) is \(\exists x\in D,\,P(x)\).

  • TRUE if \(P(x)\) is true for at least one \(x\in D\).
  • FALSE if \(P(x)\) is false for all \(x\in D\).
Examples
  • \(\exists x\in\mathbb{Z},\,x^2=x\) is TRUE (witness: \(x=0\), \(0^2=0\)).
  • \(\exists x\in\{-1,2,3\},\,x^2=x\) is FALSE (none of \(1,4,9\) equals \(-1,2,3\)).
  • "The integer 24 can be written as a sum of two integers": \(\exists m,n\in\mathbb{Z}(24=m+n)\). TRUE.

Translating from English — Key Patterns

Critical Pattern: ∀ with → vs ∧
StatementCorrect formWrong form
"Every student in class has studied calculus"
(domain = all university students \(U\))		\(\forall x\in U,\,Q(x)\to P(x)\)\(\forall x\in U,\,Q(x)\wedge P(x)\) ✗
"Some student in class has not studied calculus"
(domain = all university students)		\(\exists x\in U,\,Q(x)\wedge\neg P(x)\)\(\exists x\in U,\,Q(x)\to\neg P(x)\) ✗

Rule: \(\forall\) pairs with \(\to\); \(\exists\) pairs with \(\wedge\).

Common mistake

\(\forall x\in U,\,Q(x)\wedge P(x)\) says "every person in the university is a student in the class AND has studied calculus." This makes the statement false as soon as any non-class-member exists in \(U\).

More Translation Examples
  • "All mathematicians wear glasses": \(\forall x\in M,\,G(x)\) or \(\forall x\in H,\,P(x)\to G(x)\)
  • "No mathematician wears glasses": \(\forall x\in M,\,\neg G(x)\)
  • "At least one mathematician does not wear glasses": \(\exists x\in M,\,\neg G(x)\)
  • "Every even integer is divisible by 2": \(\forall n\in\mathbb{Z},\,E(n)\to D(n)\)

Negating Quantified Expressions

Theorem

\[\neg\forall x\,P(x)\equiv\exists x\,\neg P(x)\]

\[\neg\exists x\,P(x)\equiv\forall x\,\neg P(x)\]

Section 1.4 — Nested Quantifiers

Definition — Nested Quantifiers

Two quantifiers are nested if one is within the scope of the other, e.g. \(\forall x\,\exists y\,(x+y=0)\). Everything within the scope of a quantifier can be treated as a propositional function of the outer variable.

Examples (domain \(\mathbb{R}\))
  • \(\forall x\,\exists y\,(x+y=0)\): TRUE — for every real \(x\), take \(y=-x\).
  • \(\forall x\,\forall y\,(x+y=y+x)\): TRUE — commutativity of addition.
  • \(\forall x\,\exists y\,(xy=1)\): FALSE — fails when \(x=0\), no \(y\) exists.

Order of Quantifiers

Key: Changing order CAN change meaning
  • "Everyone loves someone": \(\forall x\,\exists y\,P(x,y)\) — for each person, there is (possibly different) someone they love.
  • "There is someone loved by everyone": \(\exists y\,\forall x\,P(x,y)\) — a single \(y\) loved by all.

These are different. The second is stronger. (If domain is 3 people \(a,b,c\) where \(a\) loves \(b\), \(b\) loves \(c\), \(c\) loves \(a\): first is true, second is false.)

Section 1.5 — Valid and Invalid Arguments

Definition — Argument Form

An argument form is a sequence of propositions \(p_1,p_2,\ldots,p_n,c\). The propositions \(p_1,\ldots,p_n\) are premises / assumptions / hypotheses; \(c\) is the conclusion. An argument form is valid when, whenever all premises are true, the conclusion is also true.

Testing Validity (Truth-Table Method)
  1. Identify premises and conclusion.
  2. Construct a truth table.
  3. If any row has all premises true and conclusion false → INVALID. Otherwise → VALID.

Rules of Inference

NameFormDescription
Modus Ponens
(Affirming)		\(p\to q,\;p\;\therefore q\)If the implication and the hypothesis hold, the conclusion holds.
Modus Tollens
(Denying)		\(p\to q,\;\neg q\;\therefore\neg p\)If the implication holds and the conclusion is false, the hypothesis is false.
Specialisation\(p\wedge q\;\therefore p\) or \(q\)From a conjunction, extract either conjunct.
Generalisation\(p\;\therefore p\vee q\)Any proposition implies any disjunction containing it.
Elimination\(p\vee q,\;\neg p\;\therefore q\)If a disjunct is false, the other must be true.
Transitivity\(p\to q,\;q\to r\;\therefore p\to r\)Chaining implications.
Proof by Cases\(p\vee q,\;p\to r,\;q\to r\;\therefore r\)If each case leads to the same conclusion, that conclusion holds.
Proof by Contradiction\(\neg p\to\mathbf{C}\;\therefore p\)If an assumption leads to a contradiction, the assumption is false.

Common Fallacies

Fallacy 1 — Affirming the Consequent (Converse Error)

Form: \(p\to q,\;q\;\therefore p\). INVALID. Counter example: \(p=F,q=T\) — both premises true but conclusion false.

Fallacy 2 — Denying the Antecedent (Inverse Error)

Form: \(p\to q,\;\neg p\;\therefore\neg q\). INVALID. Counter example: \(p=F,q=T\) — premises hold, conclusion \(\neg q=F\).

Worked Examples

Example — Knights and Knaves

Knights always tell the truth, knaves always lie. Let \(a\): "A is a knight", \(b\): "B is a knight."

A says "B is a knight." B says "A and I are of opposite type."

A's statement translates to \(a\to b\) and \(\neg a\to\neg b\). B's statement translates to \(b\to\neg a\) and \(\neg b\to\neg a\).

Solution (chaining):

  1. \(a\to b\) (A's statement, if A is knight)
  2. \(b\to\neg a\) (B's statement, if B is knight)
  3. \(\therefore a\to\neg a\) (transitivity from 1, 2)
  4. \(\therefore\neg a\vee\neg a=\neg a\) (simplify)
  5. \(\neg a\to\neg b\) (A's statement, if A is knave)
  6. \(\therefore\neg b\) (modus ponens on 4, 5)

Both A and B are knaves (consistent with facts). ✓

Tutorial 1 — Propositional Logic

Q1. Which of these are propositions? (a) Do not go. (b) Can you answer the question? (c) There are no black flies in Maine. (d) \(4+x=5\). (e) \(2^n\ge100\). (f) The moon is made of green cheese.

  • (a) NOT a proposition — command.
  • (b) NOT a proposition — question.
  • (c) PROPOSITION — declarative sentence, either T or F.
  • (d) NOT a proposition — contains variable \(x\); it is a predicate.
  • (e) NOT a proposition — contains variable \(n\); it is a predicate.
  • (f) PROPOSITION — declarative sentence (happens to be false).

Q2. Let \(p\): "The election is decided", \(q\): "The votes have been counted." Express as English sentences: (a) \(\neg p\wedge q\) (b) \(q\to p\) (c) \(p\leftrightarrow q\) (d) \(\neg q\vee(\neg p\wedge q)\).

  • (a) "The election is not decided and the votes have been counted."
  • (b) "If the votes have been counted, then the election is decided."
  • (c) "The election is decided if and only if the votes have been counted."
  • (d) \(\neg q\vee(\neg p\wedge q)\). Note: \(\neg q\vee(\neg p\wedge q)\equiv(\neg q\vee\neg p)\wedge(\neg q\vee q)\equiv(\neg q\vee\neg p)\wedge\mathbf{T}\equiv\neg p\vee\neg q\equiv\neg(p\wedge q)\). So: "It is not the case that both the election is decided and the votes have been counted."

Q3. Are the following conditional propositions true or false? (a) If \(1+1=3\), then unicorns exist. (b) If \(1+1=3\), then dogs can fly. (c) If \(1+1=2\), then dogs can fly. (d) If \(2+2=4\), then \(1+2=3\).

  • (a) \(p=F,q=F\Rightarrow p\to q=\)TRUE (vacuously true — false hypothesis).
  • (b) \(p=F,q=F\Rightarrow p\to q=\)TRUE (vacuously true).
  • (c) \(p=T,q=F\Rightarrow p\to q=\)FALSE (true hypothesis, false conclusion).
  • (d) \(p=T,q=T\Rightarrow p\to q=\)TRUE.

Q4. "You will get an A in this module only if either you do every exercise in the textbook or you score at least 80 marks in the final." Let \(A\): get an A, \(E\): do every exercise, \(S\): score \(\ge80\). (a) When is the proposition true? (b) Given the proposition is true, will you get an A if you did not do exercise 5 and scored 79?

"only if" → \(A\to(E\vee S)\). False only when \(A=T\) and \(E\vee S=F\).

(a) The proposition is true in all situations except when you get an A but neither do all exercises nor score \(\ge80\).

(b) Did not do exercise 5 ⇒ \(E=F\). Scored 79 ⇒ \(S=F\). So \(E\vee S=F\). Given the proposition \(A\to(E\vee S)\) is true and \(E\vee S=F\): by modus tollens, \(A=F\). You would NOT get an A.

Tutorial 2 — Predicates & Quantifiers

Q1. Translate into English where \(R(x)\): "\(x\) is a rabbit", \(H(x)\): "\(x\) hops", domain = all animals. (a) \(\forall x(R(x)\to H(x))\) (b) \(\forall x(R(x)\wedge H(x))\) (c) \(\exists x(R(x)\to H(x))\) (d) \(\exists x(R(x)\wedge H(x))\)

  • (a) "Every rabbit hops." (correct universal translation with \(\to\))
  • (b) "Every animal is both a rabbit and hops." (WRONG for "every rabbit hops" — this claims ALL animals are rabbits)
  • (c) "There is an animal such that if it is a rabbit, then it hops." (vacuously true for any non-rabbit — not a useful statement)
  • (d) "There is an animal that is a rabbit and hops." = "Some rabbit hops."

Q2. \(Q(x,y)\): "If \(x<y\) then \(x^2<y^2\)", domain \(\{1,\pm2\}\). (a) Why is \(Q(-2,1)\) false and \(Q(1,2)\) true? (b) Find all \((x,y)\) for which \(Q(x,y)\) is true.

(a) \(Q(-2,1)\): hypothesis \(-2<1\) is TRUE; conclusion \(4<1\) is FALSE → conditional is FALSE. \(Q(1,2)\): hypothesis \(1<2\) TRUE; conclusion \(1<4\) TRUE → TRUE.

(b) \(Q(x,y)\) is false only when \(x<y\) AND \(x^2\ge y^2\). Pairs with \(x<y\): \((-2,-1): 4\ge1\to F\); \((-2,1): 4\ge1\to F\); \((-2,2): 4\ge4\to F\); \((-1,1): 1\ge1\to F\); \((-1,2): 1<4\to T\); \((1,2): 1<4\to T\). All pairs with \(x\ge y\): vacuously true. So \(Q(x,y)\) is true for: \((-1,2),(1,2)\), and all pairs where \(x\ge y\).

Tutorial 3 — Arguments & Proofs

Q1. Use a truth table to determine whether each argument form is valid. (a) \(p\to r,\;q\to r\;\therefore\;p\vee q\to r\). (b) \(p\to q\vee r,\;\neg q\vee\neg r\;\therefore\;\neg p\vee\neg r\). (c) \(p\vee q,\;p\to r,\;q\to s\;\therefore\;r\vee s\).

(a) VALID. Suppose \(p\vee q\to r\) is false: \(p\vee q=T\) and \(r=F\). If \(r=F\): \(p\to r=\neg p\) so \(p=F\); \(q\to r=\neg q\) so \(q=F\). But then \(p\vee q=F\), contradicting \(p\vee q=T\). ✓

(b) INVALID. Counter example: \(p=T,q=F,r=T\): \(p\to q\vee r=T\to T=T\); \(\neg q\vee\neg r=T\vee F=T\); but \(\neg p\vee\neg r=F\vee F=F\). Both premises true, conclusion false. ✗

(c) VALID. Suppose \(r\vee s\) is false: \(r=F,s=F\). Then \(p\to r=\neg p\Rightarrow p=F\) and \(q\to s=\neg q\Rightarrow q=F\). But premise says \(p\vee q=T\). Contradiction. ✓

Q2. Use valid argument forms to derive \(\neg q\) from: \(\neg p\vee q\to r\), \(s\vee\neg q\), \(\neg t\), \(p\to t\), \(\neg p\wedge r\to\neg s\).

1. \(p\to t\) (hyp 4); \(\neg t\) (hyp 3) → \(\neg p\) by Modus Tollens
2. \(\neg p\) → \(\neg p\vee q=T\) by Generalisation
3. \((\neg p\vee q)\to r\) (hyp 1); \(\neg p\vee q=T\) (step 2) → \(r\) by Modus Ponens
4. \(\neg p\) (step 1) and \(r\) (step 3) → \(\neg p\wedge r\) by Conjunction
5. \((\neg p\wedge r)\to\neg s\) (hyp 5); \(\neg p\wedge r\) (step 4) → \(\neg s\) by Modus Ponens
6. \(s\vee\neg q\) (hyp 2); \(\neg s\) (step 5) → \(\neg q\) by Elimination ✓
