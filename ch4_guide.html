4.1 The Principle of Mathematical Induction

What is induction?

Mathematical induction is a proof technique used to show that a propositional function \(P(n)\) is true for all \(n \in \mathbb{Z}^+\) (or all integers from some base). It is not empirical induction — it is a logically rigorous deductive proof.

Two variants covered

  • Standard (weak) induction: Assume \(P(k)\) to prove \(P(k+1)\).
  • Strong induction: Assume \(P(1), P(2), \ldots, P(k)\) to prove \(P(k+1)\). Required when \(P(k+1)\) depends on more than just \(P(k)\) (e.g., Fibonacci recurrences, 3-term sequences).
Common trap

Checking many cases never proves a universal statement. Induction is needed. Conversely, finding even one counterexample disproves a universal claim.

Principle of Mathematical Induction

To prove \(\forall n \in \mathbb{Z}^+\,(P(n))\) where \(P(n)\) is a propositional function, we complete two steps:

Basis step: Verify that \(P(1)\) is true.

Inductive step: Show that \(\forall k \in \mathbb{Z}^+\,(P(1) \wedge P(2) \wedge \cdots \wedge P(k) \to P(k+1))\) is true.

The induction hypothesis

In the inductive step we assume that \(P(1), \ldots, P(k)\) are all true. This assumption is called the induction hypothesis. We then use it to prove \(P(k+1)\).

Not circular

We are NOT asserting \(P(k)\) holds for all \(k\). We are proving: "if \(P(1),\ldots,P(k)\) then \(P(k+1)\)". The two steps together give the chain:
\(P(1)\) (basis) \(\Rightarrow P(2) \Rightarrow P(3) \Rightarrow \cdots\)

Chain of implications built by induction

\[P(1) \quad \text{(basis)}\]

\[P(1) \Rightarrow P(2)\]

\[P(1) \wedge P(2) \Rightarrow P(3)\]

\[P(1) \wedge P(2) \wedge P(3) \Rightarrow P(4)\]

\[\vdots\]

Eventually we get \(P(5), P(6), \ldots\) for all positive integers.

When to use strong induction

Use strong induction when \(P(k+1)\) requires more than one previous case. Examples:

  • Fibonacci recurrence \(F_{k+1} = F_k + F_{k-1}\) — needs two cases.
  • Three-term recurrences \(h_{k+1} = h_k + h_{k-1} + h_{k-2}\) — needs three base cases.
  • Binary representation — case-splits on even/odd using \(k/2 \leq k\).
Proof template
  1. State: "Let \(P(n)\) be the proposition [statement]."
  2. Basis step: Show \(P(1)\) [or \(P(0)\), \(P(n_0)\)] is true by direct computation.
  3. Inductive step: State "Assume \(P(1),\ldots,P(k)\) are true [induction hypothesis]." Then show \(P(k+1)\) follows.
  4. Conclude: "Therefore \(P(n)\) is true for all \(n \in \mathbb{Z}^+\) by mathematical induction."

4.2 Lecture Examples

Example 1 — Sum formula

Example

Prove that \(\forall n \in \mathbb{Z}^+,\; \displaystyle\sum_{i=1}^{n} i = \frac{n(n+1)}{2}.\)

Let \(P(n)\) be the proposition \(\sum_{i=1}^n i = \tfrac{n(n+1)}{2}\).

Basis step: \(P(1)\): \(\sum_{i=1}^1 i = 1 = \tfrac{1 \cdot 2}{2}\). True.

Inductive step: Assume \(P(1),\ldots,P(k)\) are true, i.e. \[\sum_{i=1}^j i = \frac{j(j+1)}{2} \quad \text{for } j=1,2,\ldots,k.\] We use \(P(k)\) to prove \(P(k+1)\): \[\sum_{i=1}^{k+1} i = \left(\sum_{i=1}^k i\right) + (k+1) = \frac{k(k+1)}{2} + (k+1) = \frac{k(k+1)+2(k+1)}{2} = \frac{(k+1)(k+2)}{2}.\] This is exactly \(P(k+1)\). \(\square\)

Example 2 — Exponential inequality

Example

Prove that \(n \lt 2^n\) for all \(n \in \mathbb{Z}^+\).

Basis: \(P(1)\): \(1 \lt 2^1 = 2\). True.

Inductive step: Assume \(P(k)\): \(k \lt 2^k\). Then: \[k+1 \lt 2^k + 1 \leq 2^k + 2^k = 2^{k+1}.\] (We used \(1 \leq 2^k\) for all \(k \geq 1\).) \(\square\)

Example 3 — Harmonic numbers

Example (uses strong induction; base is \(n=0\))

The harmonic numbers are \(H_j = 1 + \tfrac{1}{2} + \cdots + \tfrac{1}{j}\) for \(j \in \mathbb{Z}^+\).

Prove that \(H_{2^n} \geq 1 + \tfrac{n}{2}\) for all \(n \in \mathbb{Z}^*\).

Basis: \(P(0)\): \(H_{2^0} = H_1 = 1 \geq 1 + \tfrac{0}{2} = 1\). True.

Inductive step: Assume \(P(0),\ldots,P(k)\). Then: \[H_{2^{k+1}} = H_{2^k} + \left(\frac{1}{2^k+1}+\frac{1}{2^k+2}+\cdots+\frac{1}{2^{k+1}}\right).\] There are \(2^k\) new terms, each \(\geq \tfrac{1}{2^{k+1}}\), so their sum \(\geq 2^k \cdot \tfrac{1}{2^{k+1}} = \tfrac{1}{2}\). Using \(P(k)\): \(H_{2^k} \geq 1 + \tfrac{k}{2}\). Therefore: \[H_{2^{k+1}} \geq \left(1+\frac{k}{2}\right)+\frac{1}{2} = 1+\frac{k+1}{2}.\] This is \(P(k+1)\). \(\square\)

Example 4 — Checkerboard tiling

Example (geometric / structural)

Prove that for any \(n \geq 1\), if one square is removed from a \(2^n \times 2^n\) checkerboard, the remaining squares can be covered by L-trominoes (L-shaped pieces covering exactly 3 squares).

Basis: \(P(1)\): A \(2 \times 2\) board with one square removed leaves 3 squares forming an L-tromino. True.

Inductive step: Assume \(P(k)\). Consider a \(2^{k+1} \times 2^{k+1}\) board with one square removed.

  • Divide it into four \(2^k \times 2^k\) quadrants.
  • The removed square is in one quadrant (say the top-left). The other three quadrants are intact.
  • Place one L-tromino at the centre of the board, with one square in each of the three intact quadrants' corners.
  • Now each quadrant is a \(2^k \times 2^k\) board with exactly one square missing. By \(P(k)\), each can be tiled.

Hence \(P(k+1)\) holds. \(\square\)

4.3 Theorems

Theorem — Number of subsets

Theorem

A set with \(n\) elements has \(2^n\) subsets.

Proof. Let \(Q(n)\) be the proposition \(|\mathcal{P}(X)| = 2^n\) for any set \(X\) with \(|X|=n\).

Basis: \(Q(0)\): \(\emptyset\) has exactly one subset (\(\emptyset\) itself), and \(2^0 = 1\). True.

Inductive step: Assume \(Q(k)\). Let \(X\) be any set with \(k+1\) elements. Pick any \(a \in X\) and let \(Y = X \setminus \{a\}\), so \(|Y|=k\). Subsets of \(X\) split into two types:

  • Those not containing \(a\): precisely the subsets of \(Y\); there are \(2^k\) of these (by \(Q(k)\)).
  • Those containing \(a\): each corresponds to a subset of \(Y\) with \(a\) added; there are also \(2^k\) of these.

Total: \(|\mathcal{P}(X)| = 2^k + 2^k = 2^{k+1}\). \(\square\)

Theorem — Sum of a geometric progression

Theorem (GP sum)

For all \(n \in \mathbb{Z}^*\) and all real \(r \neq 1\): \[\sum_{i=0}^{n} r^i = \frac{r^{n+1}-1}{r-1}.\]

Proof. Let \(P(n)\) be the above equality.

Basis (\(n=0\)): LHS \(= r^0 = 1\). RHS \(= \tfrac{r^1-1}{r-1} = 1\). True.

Inductive step: Assume \(P(k)\): \(\sum_{i=0}^k r^i = \tfrac{r^{k+1}-1}{r-1}\). Then: \[\sum_{i=0}^{k+1} r^i = \sum_{i=0}^k r^i + r^{k+1} = \frac{r^{k+1}-1}{r-1} + r^{k+1} = \frac{r^{k+1}-1 + r^{k+1}(r-1)}{r-1} = \frac{r^{k+2}-1}{r-1}.\] This is \(P(k+1)\). \(\square\)

Common mistakes in induction proofs

Missing / wrong basis step

Changing the base to \(n=0\) or \(n=2\) without noticing, or skipping it entirely, invalidates the proof.

Using what you want to prove

Writing \(P(k+1)\) on both sides without actually deriving it from \(P(k)\) is circular.

Strong induction with insufficient base cases

A \(d\)-term recurrence requires \(d\) base cases. A 2-term recurrence needs \(P(1)\) and \(P(2)\) both verified.

Off-by-one in limits

When manipulating \(\sum_{i=1}^{k+1} = \sum_{i=1}^k + (k+1)\text{th term}\), ensure the last term is correct.

4.4 Fibonacci Numbers & Lamé's Theorem

Fibonacci numbers

Definition

The Fibonacci sequence \(F_0, F_1, F_2, \ldots\) is defined by: \[F_0 = 0,\quad F_1 = 1,\quad F_{n+1} = F_n + F_{n-1} \quad \text{for } n \geq 1.\]

First values: \(F_0=0,\; F_1=1,\; F_2=1,\; F_3=2,\; F_4=3,\; F_5=5,\; F_6=8,\; F_7=13,\ldots\)

Theorem — Fibonacci growth bound

Theorem

Let \(\alpha = \dfrac{1+\sqrt{5}}{2} \approx 1.618\). Then \(F_n > \alpha^{n-2}\) for all \(n \geq 3\).

Key algebraic identity: \(\alpha^2 = \dfrac{(1+\sqrt{5})^2}{4} = \dfrac{3+\sqrt{5}}{2} = 1 + \alpha.\)

Proof (strong induction with two base cases).

Basis: \(P(3)\): \(F_3=2 > \alpha^1 = \alpha \approx 1.618\). True. \(P(4)\): \(F_4=3 > \alpha^2 = 1+\alpha \approx 2.618\). True.

We need both \(P(3)\) and \(P(4)\) because the recurrence uses the two previous terms.

Inductive step: Assume \(P(n)\) for \(n = 3,\ldots,k\) (where \(k \geq 4\)). Then: \[F_{k+1} = F_k + F_{k-1} > \alpha^{k-2} + \alpha^{k-3} = \alpha^{k-3}(\alpha + 1) = \alpha^{k-3} \cdot \alpha^2 = \alpha^{k-1}.\] This is \(P(k+1)\). \(\square\)

Lame's theorem (Euclidean algorithm efficiency)

Lame's theorem

Let \(a, b\) be positive integers with \(a \geq b\). The number of divisions used by the Euclidean algorithm to compute \(\gcd(a,b)\) is at most \(5 \times\) the number of decimal digits in \(b\).

Example: \(\gcd(1{,}034{,}578{,}929{,}341,\; 2018)\): at most \(5 \times 4 = 20\) steps.

Proof sketch

Let the Euclidean algorithm produce remainders \(r_1 = a,\; r_2 = b,\; r_3,\;\ldots,\; r_n,\; r_{n+1} = 0\), so \(\gcd = r_n\). Show that \(r_i \geq F_{n+3-i}\) for \(i = 2,\ldots,n+1\) by backward induction:

  • \(r_{n+1} = 0\) and \(r_n = \gcd \geq 1 = F_2\).
  • \(r_{n-1} \geq r_n + r_{n+1} \geq F_2 + F_1 = F_3 = 2\), since \(r_{n-1} = r_n q + r_{n+1}\) with \(q \geq 1\).
  • Continuing: \(b = r_2 \geq F_{n+1} > \alpha^{n-1}\).

Let \(m\) = number of digits of \(b\). Then \(\log_{10} \alpha > 1/5\), so: \[m \geq \log_{10}(b+1) \geq (n-1)\log_{10} \alpha > \frac{n-1}{5}.\] Hence \(n-1 \lt 5m\), and since \(n\) is an integer, \(n \leq 5m\). \(\square\)

Q1(a). Prove \(\displaystyle\sum_{i=1}^{n+1} i \cdot 2^i = n \cdot 2^{n+2} + 2\) for all \(n \in \mathbb{Z}^*\).

Let \(P(n)\) be \(\sum_{i=1}^{n+1} i \cdot 2^i = n \cdot 2^{n+2} + 2\).

Basis (\(n=0\)): LHS \(= \sum_{i=1}^1 i \cdot 2^i = 1 \cdot 2 = 2\). RHS \(= 0 \cdot 4 + 2 = 2\). True.

Inductive step: Assume \(P(k)\): \(\sum_{i=1}^{k+1} i \cdot 2^i = k \cdot 2^{k+2} + 2\). Then: \[\sum_{i=1}^{k+2} i \cdot 2^i = \sum_{i=1}^{k+1} i \cdot 2^i + (k+2) \cdot 2^{k+2} = k \cdot 2^{k+2} + 2 + (k+2) \cdot 2^{k+2}\] \[= 2^{k+2}(k + k + 2) + 2 = 2^{k+2} \cdot 2(k+1) + 2 = (k+1) \cdot 2^{k+3} + 2.\] This is \(P(k+1)\). \(\square\)

Q1(b). Prove \(6 \mid 7^n - 1\) for all \(n \in \mathbb{Z}^*\).

Basis (\(n=0\)): \(7^0 - 1 = 0\), and \(6 \mid 0\). True.

Inductive step: Assume \(6 \mid 7^k - 1\). Then: \[7^{k+1} - 1 = 7 \cdot 7^k - 1 = 7(7^k - 1) + 6.\] Since \(6 \mid 7^k - 1\) (IH) and \(6 \mid 6\), we have \(6 \mid 7^{k+1} - 1\). \(\square\)

Q1(c). Prove \(1 + nx \leq (1+x)^n\) for all integers \(n \geq 2\) and all real \(x > -1\) (Bernoulli's inequality).

Basis (\(n=2\)): \((1+x)^2 = 1 + 2x + x^2 \geq 1 + 2x\) since \(x^2 \geq 0\). True.

Inductive step: Assume \(1 + kx \leq (1+x)^k\) with \(k \geq 2\). Since \(x > -1\), we have \(1+x > 0\), so: \[(1+x)^{k+1} = (1+x)^k \cdot (1+x) \geq (1+kx)(1+x) = 1 + kx + x + kx^2 = 1 + (k+1)x + kx^2.\] Since \(kx^2 \geq 0\), we get \((1+x)^{k+1} \geq 1+(k+1)x\). \(\square\)

Q2. Prove by induction that \(n^3 + 11n\) is divisible by 3 for all \(n \in \mathbb{Z}^+\).

Basis (\(n=1\)): \(1 + 11 = 12 = 3 \times 4\). True.

Inductive step: Assume \(3 \mid k^3 + 11k\). Then: \[(k+1)^3 + 11(k+1) = k^3 + 3k^2 + 3k + 1 + 11k + 11 = (k^3 + 11k) + 3(k^2 + k + 4).\] By the IH, \(3 \mid k^3 + 11k\). Clearly \(3 \mid 3(k^2 + k + 4)\). Hence \(3 \mid (k+1)^3 + 11(k+1)\). \(\square\)

Q3. Let \(h_0=1, h_1=2, h_2=3\) and \(h_k = h_{k-1}+h_{k-2}+h_{k-3}\) for \(k \geq 3\). Prove \(h_n \leq 3^n\) for all \(n \geq 0\).

Three base cases (recurrence uses 3 previous terms):

  • \(h_0 = 1 \leq 1 = 3^0\). True.
  • \(h_1 = 2 \leq 3 = 3^1\). True.
  • \(h_2 = 3 \leq 9 = 3^2\). True.

Inductive step (strong): Assume \(h_j \leq 3^j\) for \(j = 0,1,\ldots,k\) where \(k \geq 2\). Then: \[h_{k+1} = h_k + h_{k-1} + h_{k-2} \leq 3^k + 3^{k-1} + 3^{k-2} = 3^{k-2}(9+3+1) = 13 \cdot 3^{k-2}.\] Since \(13 \leq 27 = 3^3\), we get \(h_{k+1} \leq 3^3 \cdot 3^{k-2} = 3^{k+1}\). \(\square\)

Q4. What is wrong with the following proof that \(2^n = 1\) for all \(n \in \mathbb{Z}^*\)?
Basis: \(2^0 = 1\). Inductive step: Assume \(2^j = 1\) for \(j = 0,1,\ldots,k\). Then \(2^{k+1} = \dfrac{2^k \cdot 2^k}{2^{k-1}} = \dfrac{1 \cdot 1}{1} = 1\).

Flaw: The inductive step substitutes \(2^{k-1} = 1\) (from the IH). But when \(k = 0\), this requires \(2^{-1} = 1\), which is false, and \(k = -1\) is not a non-negative integer covered by the IH.

The step fails for \(k = 0\) (the transition from \(n=0\) to \(n=1\)), so the chain is broken immediately. The algebraic manipulation requires \(k \geq 1\), meaning the step from \(2^0\) to \(2^1\) is never justified.

Moral

Always verify the inductive step is valid at the smallest value of \(k\). Here the IH covers \(j=0\) only, but the algebra silently uses \(j=-1\).

Q5. For which positive integers \(n\) is \(2^n > n^2 + n\) true? Make an educated guess, then prove it by induction.

Checking small values:

  • \(n=1\): \(2 > 2\)? No.
  • \(n=2\): \(4 > 6\)? No.
  • \(n=3\): \(8 > 12\)? No.
  • \(n=4\): \(16 > 20\)? No.
  • \(n=5\): \(32 > 30\)? Yes!

Conjecture: True for all \(n \geq 5\).

Basis (\(n=5\)): \(32 > 30\). True.

Inductive step: Assume \(2^k > k^2 + k\) for some \(k \geq 5\). Then: \[2^{k+1} = 2 \cdot 2^k > 2(k^2+k) = 2k^2 + 2k.\] We need to show \(2k^2 + 2k > (k+1)^2 + (k+1) = k^2 + 3k + 2\), i.e., \(k^2 - k - 2 > 0\), i.e., \((k-2)(k+1) > 0\). For \(k \geq 5\): \(k-2 \geq 3 > 0\) and \(k+1 > 0\), so the product is positive. \(\square\)

Q6. Prove by induction that every positive integer can be written as a sum of distinct non-negative integer powers of 2.

Strong induction. Let \(P(n)\): \(n\) is a sum of distinct powers of 2.

Basis (\(n=1\)): \(1 = 2^0\). True.

Inductive step: Assume \(P(m)\) for all \(1 \leq m \leq k\). Consider \(k+1\).

  • Case 1: \(k+1\) is even. Then \(\frac{k+1}{2}\) is a positive integer \(\leq k\). By IH, \(\frac{k+1}{2} = 2^{i_1}+\cdots+2^{i_r}\) (distinct powers). Then \(k+1 = 2^{i_1+1}+\cdots+2^{i_r+1}\), distinct powers (all \(\geq 1\)). \(\checkmark\)
  • Case 2: \(k+1\) is odd. Then \(k = k+1-1\) is even, so \(k/2 \leq k\) and by IH \(k = 2^{j_1}+\cdots+2^{j_r}\) (distinct). Since \(k\) is even, all \(j_i \geq 1 > 0\). Thus \(k+1 = 2^0 + 2^{j_1}+\cdots+2^{j_r}\), distinct powers. \(\checkmark\)

\(\square\)

Q7. Prove by induction on \(n\): \(\forall n \in \mathbb{Z}_{\geq 8}\; (\exists x, y \in \mathbb{Z}_{\geq 0})\; (n = 3x + 5y)\).

Base cases: \(8 = 3+5\), \(9 = 3\times3\), \(10 = 2\times5\). True for \(n=8,9,10\).

Inductive step (strong, \(k \geq 10\)): Assume \(P(m)\) for all \(8 \leq m \leq k\). Then \(k+1 - 3 = k-2 \geq 8\). By IH, \(k-2 = 3x+5y\) for some \(x,y \geq 0\). Therefore \(k+1 = 3(x+1)+5y\). \(\square\)

Q8. Show by induction: \(F_{n+1}^2 - F_{n+1}F_n - F_n^2 = (-1)^n\) for every \(n \in \mathbb{Z}_{\geq 0}\) (Cassini-like identity).

Basis (\(n=0\)): \(F_1^2 - F_1 F_0 - F_0^2 = 1 - 0 - 0 = 1 = (-1)^0\). True.

Inductive step: Assume \(F_{k+1}^2 - F_{k+1}F_k - F_k^2 = (-1)^k\). Use \(F_{k+2} = F_{k+1}+F_k\): \[F_{k+2}^2 - F_{k+2}F_{k+1} - F_{k+1}^2\] \[= (F_{k+1}+F_k)^2 - (F_{k+1}+F_k)F_{k+1} - F_{k+1}^2\] \[= F_{k+1}^2 + 2F_{k+1}F_k + F_k^2 - F_{k+1}^2 - F_k F_{k+1} - F_{k+1}^2\] \[= F_{k+1}F_k + F_k^2 - F_{k+1}^2 = -(F_{k+1}^2 - F_{k+1}F_k - F_k^2) = -(-1)^k = (-1)^{k+1}.\] \(\square\)

Q10. Define set \(S\) recursively: (1) \(2 \in S\); (2) if \(x \in S\) then \(3x \in S\) and \(x^2 \in S\); (3) minimality clause. Which of \(0,\; 6,\; 15,\; 16,\; 36\) are in \(S\)? Which are not?

Build \(S\) from 2:

  • \(2 \in S\) (basis).
  • \(3\times2 = 6 \in S\); \(2^2 = 4 \in S\).
  • From 4: \(3\times4=12\in S\); \(4^2=16\in S\).
  • From 6: \(3\times6=18\in S\); \(6^2=36\in S\).

Results:

  • \(0\): Not in \(S\). Every element of \(S\) is \(\geq 2\) (operations \(\times3\) and \((\cdot)^2\) on positive integers \(\geq 2\) keep values \(\geq 6\) or \(\geq 4\)).
  • \(6\): In \(S\). \(6 = 3\times2\).
  • \(15\): Not in \(S\). \(15/3=5\), and 5 is not reachable (\(\sqrt{5}\) irrational, \(5/3\) not integer). By the minimality clause, 15 is excluded.
  • \(16\): In \(S\). \(16 = 4^2 = (2^2)^2\).
  • \(36\): In \(S\). \(36 = 6^2 = (3\times2)^2\).
