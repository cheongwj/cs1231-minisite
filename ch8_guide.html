8.1–8.2 Relations

Definition — Relation

Let \(A, B\) be sets. A relation \(R\) from \(A\) to \(B\) is a subset of \(A \times B\).

We write \(x\,R\,y\) iff \((x,y) \in R\), and \(x\,\not R\,y\) iff \((x,y) \notin R\).

A relation on \(A\) is a relation from \(A\) to \(A\), i.e. \(R \subseteq A \times A\).

Examples
  • \(A=\{0,1,2\}\), \(B=\{1,2,3\}\), \(x\,R\,y\) iff \(x\lt y\): \(R=\{(0,1),(0,2),(0,3),(1,2),(1,3),(2,3)\}\). So \(0\,R\,1\), but \(2\,\not R\,2\).
  • \(E=\{(x,y):x\equiv y\pmod{2}\}\) on \(\mathbb{Z}\times\mathbb{Z}\): \(2\,E\,6\), \(3\,E\,{-11}\), but \(2\,\not E\,7\).
  • Bit strings of length 6; \(x\,R\,y\) if first 4 bits match: \(101000\,R\,101011\), but \(101111\,\not R\,101011\).
  • For any function \(f:A\to B\), the graph \(\Gamma(f)=\{(a,f(a))\mid a\in A\}\) is a relation. Functions are special relations.
Definition — Domain & Range

Domain of \(R\) \(=\{a\in A\mid\exists b\in B,\,a\,R\,b\}\).

Range of \(R\) \(=\{b\in B\mid\exists a\in A,\,a\,R\,b\}\).

When \(R=\Gamma(f)\), domain and range of \(R\) equal those of \(f\).

Arrow Diagram

Arrow Diagram

A relation \(R\) is represented by an arrow diagram: draw an arrow from \(a\) to \(b\) whenever \((a,b)\in R\).

Example

\(R=\{(1,x),(1,y),(2,x),(2,y),(2,z),(3,z),(4,z)\}\) from \(A=\{1,2,3,4\}\) to \(B=\{x,y,z\}\). In the arrow diagram, 2 has arrows to all three elements of \(B\); 1 has arrows to \(x\) and \(y\); 3 and 4 each go only to \(z\).

Definition — Inverse Relation

The inverse of \(R\) is \(R^{-1}=\{(b,a)\in B\times A\mid a\,R\,b\}\).

  • \(\forall a\in A\,\forall b\in B\;(a\,R\,b\Leftrightarrow b\,R^{-1}\,a)\).
  • Arrow diagram of \(R^{-1}\) = reverse all arrows of \(R\).
  • \((\Gamma(f))^{-1}\) is a function \(g:B\to A\) iff \(f\) is bijective, in which case \(g=f^{-1}\).

8.3 Equivalence Relations

Four Key Properties of Relations on \(A\)
  • Reflexive: \(\forall x\in A,\;(x,x)\in R\).
  • Symmetric: \(\forall x,y\in A,\;(x,y)\in R\Rightarrow(y,x)\in R\).
  • Antisymmetric: \(\forall x,y\in A,\;(x,y)\in R\text{ and }(y,x)\in R\Rightarrow x=y\). (At most one of \((x,y)\), \((y,x)\) can be in \(R\) for \(x\ne y\).)
  • Transitive: \(\forall x,y,z\in A,\;(x,y)\in R\text{ and }(y,z)\in R\Rightarrow(x,z)\in R\).

Definition — Equivalence Relation

A relation \(R\) on \(A\) is an equivalence relation if it is reflexive, symmetric, and transitive.

If \((a,b)\in R\), we say \(a\) is equivalent to \(b\).

Definition — Equivalence Class

The equivalence class of \(a\in A\) under equivalence relation \(R\) is:

\[[a]_R=\{x\in A\mid(a,x)\in R\}\]

The set of all equivalence classes is \(A/R=\{[a]_R\mid a\in A\}\subseteq\mathcal{P}(A)\).

Example 1 — Relation \(S\) on \(\mathbb{R}\): \((a,b)\in S\) iff \(a-b\in\mathbb{Z}\)

Reflexive: \(a-a=0\in\mathbb{Z}\). ✓

Symmetric: \(a-b\in\mathbb{Z}\Rightarrow b-a=-(a-b)\in\mathbb{Z}\). ✓

Transitive: \(a-b,b-c\in\mathbb{Z}\Rightarrow a-c=(a-b)+(b-c)\in\mathbb{Z}\). ✓

Classes: \([a]=\{n+a\mid n\in\mathbb{Z}\}\). So \(S/\mathbb{R}=\{[a]\mid 0\le a\lt1\}\) — one class per fractional part.

Example 2 — Congruence mod \(m\): \(R=\{(a,b)\mid a\equiv b\pmod{m}\}\)

Recall \(a\equiv b\pmod{m}\) iff \(m\mid(a-b)\).

Reflexive: \(m\mid 0\). ✓  Symmetric: \(m\mid(a-b)\Rightarrow m\mid(b-a)\). ✓  Transitive: \(m\mid(a-b)\) and \(m\mid(b-c)\Rightarrow m\mid(a-c)\). ✓

Classes: \([n]=\{n+km\mid k\in\mathbb{Z}\}=[r]\) where \(r=n\bmod m\). Exactly \(m\) classes: \([0],[1],\ldots,[m-1]\).

\(m=2\): \([0]=\{2k\}\) (even), \([1]=\{1+2k\}\) (odd).  \(m=3\): \([0]=\{3k\}\), \([1]=\{3k+1\}\), \([2]=\{3k+2\}\).

Example 3 — Non-example: Divides on \(\mathbb{Z}\)

The divides relation is not an equivalence relation: it is not symmetric (\(2\mid 6\) but \(6\nmid 2\)).

Example 4 — Bitstring Relation \(R_n\)

Define \(R_n\) on finite bitstrings: \((s,t)\in R_n\) if \(s=t\) or both have length \(\ge n\) and their first \(n\) bits agree. Then \(R_n\) is an equivalence relation. Classes: if \(|a|\le n\), then \([a]=\{a\}\); if \(|a|\gt n\), let \(b\) = first \(n\) bits of \(a\), then \([a]=\{bx\mid x\text{ bitstring}\}\).

8.3 Partitions

Definition — Partition

A collection of nonempty subsets \(A_1,A_2,\ldots\) of \(S\) is a partition of \(S\) if:

  • \(A_1\cup A_2\cup\cdots=S\) (covers \(S\)); and
  • \(A_1,A_2,\ldots\) are mutually disjoint (\(A_i\cap A_j=\emptyset\) for \(i\ne j\)).
Examples
  • \(\{\text{primes},\text{composites},\{1\}\}\) partitions \(\mathbb{N}\).
  • \(\{1,2,3\}\) has 5 partitions: \(\{\{1\},\{2\},\{3\}\}\); \(\{\{1\},\{2,3\}\}\); \(\{\{2\},\{1,3\}\}\); \(\{\{3\},\{1,2\}\}\); \(\{\{1,2,3\}\}\).
  • The \(m\) congruence classes \([0],[1],\ldots,[m-1]\) partition \(\mathbb{Z}\).
Theorem — Equivalence Class Characterisation

For an equivalence relation \(R\) on \(A\) and \(a,b\in A\), the following are equivalent:

(i) \((a,b)\in R\)  \(\Longleftrightarrow\)  (ii) \([a]=[b]\)  \(\Longleftrightarrow\)  (iii) \([a]\cap[b]\ne\emptyset\)

(i)⇒(ii): Let \(x\in[a]\), so \((a,x)\in R\). Since \((b,a)\in R\) (symmetry) and \((a,x)\in R\), transitivity gives \((b,x)\in R\), so \(x\in[b]\). Thus \([a]\subseteq[b]\). Reversing gives \([b]\subseteq[a]\), so \([a]=[b]\).

(ii)⇒(iii): \([a]\cap[b]=[a]\ne\emptyset\) (since \(a\in[a]\) by reflexivity).

(iii)⇒(i): Let \(x\in[a]\cap[b]\). Then \((a,x)\in R\) and \((b,x)\in R\). By symmetry \((x,b)\in R\). By transitivity \((a,b)\in R\).

Corollary — Classes are Identical or Disjoint

For any \(a,b\in A\): \([a]=[b]\) or \([a]\cap[b]=\emptyset\). Equivalence classes never partially overlap.

Theorem — Partition Correspondence

The distinct equivalence classes of an equivalence relation \(R\) on \(S\) form a partition of \(S\). Conversely, every partition of \(S\) arises from some equivalence relation.

Partition from \(R\): Every \(x\in S\) is in \([x]\) (reflexivity). Distinct classes are disjoint (corollary). So distinct classes partition \(S\).

Relation from partition \(\{A_i\mid i\in I\}\): Define \((x,y)\in T\) iff \(\exists i: x,y\in A_i\).

  • Reflexive: \(\forall w,\;\exists k: w\in A_k\), so \((w,w)\in T\). ✓
  • Symmetric: immediate from definition. ✓
  • Transitive: if \(x,y\in A_i\) and \(y,z\in A_j\), then \(y\in A_i\cap A_j\), so \(i=j\) (disjoint), giving \(x,z\in A_i\). ✓

Each \(x\in A_i\) has \([x]=A_i\). ✓

8.4 Partial Orders

Definition — Partial Order & Poset

A relation \(R\) on \(A\) is a partial order if it is reflexive, antisymmetric, and transitive.

We call \((A,R)\) a partially ordered set (or poset). We write \(\preceq\) for the partial order and \(a\prec b\) when \(a\preceq b\) and \(a\ne b\).

Examples of Posets
  • \((A,\le)\) for any \(A\subseteq\mathbb{R}\).
  • \((S,|)\) for \(S\subseteq\mathbb{N}\), where \(m\,R\,n\) iff \(m\mid n\).
  • \((A,\subseteq)\) for any collection of sets.
  • If \(R\) is a partial order on \(A\), so is \(R^{-1}\).
Comparable & Noncomparable

In \((A,\preceq)\), elements \(a,b\) are comparable if \(a\preceq b\) or \(b\preceq a\); otherwise noncomparable.

  • In \((\mathbb{R},\le)\): every two reals are comparable.
  • In \((\mathcal{P}(S),\subseteq)\): \(\{a\}\) and \(\{b\}\) (\(a\ne b\)) are noncomparable.
  • In \((\mathbb{N},|)\): 2 and 3 are noncomparable.
Definitions — Extremal Elements
  • Maximal: \(a\) is maximal if \(\neg\,(\exists c: a\prec c)\) — nothing strictly above \(a\).
  • Minimal: \(a\) is minimal if \(\neg\,(\exists c: c\prec a)\) — nothing strictly below \(a\).
  • Greatest/maximum: \(a\) is greatest if \(\forall b\in A,\,b\preceq a\) — everything is below or equal to \(a\).
  • Smallest/minimum: \(a\) is smallest if \(\forall b\in A,\,a\preceq b\) — \(a\) is below or equal to everything.

Key distinction: maximal/minimal is local (no neighbour strictly beyond); greatest/smallest is global (related to every element).

Example — \((\{1,\ldots,10\},|)\)

Maximal elements: 6, 7, 8, 9, 10. No greatest element. 1 is both the smallest and the unique minimal element.

Theorem — Finite Posets Have Extremal Elements

Every nonempty finite poset has at least one minimal element and at least one maximal element.

Pick \(a_1\in S\). If not minimal, \(\exists a_2\prec a_1\). Continue to get a strictly descending chain \(a_1\succ a_2\succ\cdots\). Since \(S\) is finite and elements are distinct, the chain must terminate at a minimal element. Similarly for maximal.

Theorem — Uniqueness of Greatest/Smallest

Every nonempty poset has at most one greatest element and at most one smallest element.

Suppose \(x,y\) are both smallest. Then \(x\preceq y\) (since \(x\) is smallest) and \(y\preceq x\) (since \(y\) is smallest). Antisymmetry gives \(x=y\). Same for greatest.

8.4 Hasse Diagrams & Sorting

Definition — Hasse Diagram

The Hasse diagram of poset \((A,\preceq)\): for each cover relation \(x\prec y\) with no \(z\) such that \(x\prec z\prec y\), place \(x\) below \(y\) and draw a line from \(x\) to \(y\). Transitively implied edges are omitted.

Reading a Hasse Diagram
  • \(x\prec y\): can travel from \(x\) to \(y\) following lines upward.
  • Minimal elements: vertices with no lines going downward.
  • Maximal elements: vertices with no lines going upward.
Example — Divisors of \(\{1,2,3,5,6,10,15,30\}\)

Bottom: 1. Level 1: 2, 3, 5. Level 2: 6 \((=2\cdot3)\), 10 \((=2\cdot5)\), 15 \((=3\cdot5)\). Top: 30.

Edges drawn: \(1\text{-}2,1\text{-}3,1\text{-}5,2\text{-}6,2\text{-}10,3\text{-}6,3\text{-}15,5\text{-}10,5\text{-}15,6\text{-}30,10\text{-}30,15\text{-}30\). Edge \(1\text{-}6\) is not drawn (not a cover: \(1\prec2\prec6\)).

Example — Power Set \(\mathcal{P}(\{a,b,c\})\) under \(\subseteq\)

Top: \(\{a,b,c\}\). Level 2: \(\{a,b\},\{a,c\},\{b,c\}\). Level 1: \(\{a\},\{b\},\{c\}\). Bottom: \(\emptyset\).

Total Order, Chain, Well Order
  • Total order: \(\preceq\) on \(P\) is total if \(\forall a,b\in P\), \(a\preceq b\) or \(b\preceq a\) — every pair comparable.
  • Chain: a subset \(Q\subseteq P\) where every two elements are comparable.
  • Well order: a total order where every nonempty subset has a smallest element. \((\mathbb{Z}^+,\le)\) is a well order; \((\mathbb{Z},\le)\) is not.
Lexicographic vs Product Order
  • Lexicographic: \((x_1,\ldots,x_n)\preceq(y_1,\ldots,y_n)\) if the first differing coordinate satisfies \(x_i\prec y_i\). This is a total order.
  • Product: \((x_1,\ldots,x_n)\preceq(y_1,\ldots,y_n)\) iff \(x_i\preceq y_i\) for all \(i\). This is not total: e.g. \((1,2,3)\not\preceq(2,3,1)\) and vice versa. Product order \(\subseteq\) lexicographic order.
Topological Sorting

A total order \(\preceq\) is compatible with partial order \(R\) if \(a\,R\,b\Rightarrow a\preceq b\). The construction of such a total order is topological sorting.

Theorem — Topological Sort Exists

For any finite poset \((S,R)\), a total ordering compatible with \(R\) always exists.

Iteratively pick a minimal element from the remaining set: \(S_1=S\), pick minimal \(a_1\); \(S_2=S_1-\{a_1\}\), pick minimal \(a_2\); etc. Define \(a_i\prec a_j\) iff \(i\lt j\). This total order is compatible with \(R\): if \(a_i\,R\,a_j\) and \(i\gt j\), then \(a_j\) was removed before \(S_i\) was formed, but \(a_i\in S_i\) is minimal, yet \(a_j\,R^{-1}\,a_i\) (i.e., \(a_j\) above \(a_i\) in the original order) — contradiction.

Example — Topological Sort of \((\{1,2,3,5,6,10,15,30\},|)\)
  1. Minimal of \(\{1,2,3,5,6,10,15,30\}\): only 1. Pick \(a_1=1\).
  2. Minimal of \(\{2,3,5,6,10,15,30\}\): 2, 3, 5. Pick \(a_2=3\).
  3. Minimal of \(\{2,5,6,10,15,30\}\): 2, 5. Pick \(a_3=2\).
  4. Minimal of \(\{5,6,10,15,30\}\): 5. Pick \(a_4=5\).
  5. Minimal of \(\{6,10,15,30\}\): 6, 10, 15. Pick \(a_5=6\).
  6. Minimal of \(\{10,15,30\}\): 10, 15. Pick \(a_6=15\).
  7. Minimal of \(\{10,30\}\): 10. Pick \(a_7=10\).
  8. Pick \(a_8=30\).

Compatible total order: \(1\prec3\prec2\prec5\prec6\prec15\prec10\prec30\).

Quiz

Q1. Which properties define an equivalence relation?

A. Equivalence relation = reflexive + symmetric + transitive. B defines a partial order.

Q2. For \(R=\{(a,b):a\equiv b\pmod 3\}\) on \(\mathbb{Z}\), which is FALSE?

C is false. \([0]=\{3k\mid k\in\mathbb{Z}\}\) includes negative multiples of 3 as well (\(\ldots,-6,-3,0,3,6,\ldots\)).

Q3. \(\le\) on \(\mathbb{R}\) fails to be an equivalence relation because:

C. \(\le\) is reflexive and transitive, but not symmetric: \(1\le2\) does not imply \(2\le1\). It IS antisymmetric, making it a partial order instead.

Q4. \(R=\{(1,1),(2,2),(3,3),(1,2),(2,1)\}\) on \(\{1,2,3\}\) is:

A. Reflexive ✓; Symmetric (\((1,2)\) and \((2,1)\) both present) ✓; Transitive (check all compositions) ✓. Classes: \([1]=[2]=\{1,2\}\) and \([3]=\{3\}\).

Q5. For equivalence relation \(R\) on \(A\) and \(a,b\in A\), \((a,b)\in R\) iff:

C. By the equivalence class theorem: \((a,b)\in R\Leftrightarrow[a]=[b]\Leftrightarrow[a]\cap[b]\ne\emptyset\). A and B are the opposite; D is wrong since \(R\) is symmetric.

Q6. Which defines a partial order?

B. Partial order = reflexive + antisymmetric + transitive. D defines an equivalence relation.

Q7. Minimal elements in a Hasse diagram are those with:

A. Minimal = nothing strictly below, so no downward lines. B describes maximal elements. Confusing these is a very common error.

Q8. The divides relation \(|\) on \(\mathbb{N}\) is:

B. Reflexive (\(a\mid a\)), antisymmetric (\(a\mid b\) and \(b\mid a\Rightarrow a=b\) in \(\mathbb{N}\)), transitive ✓. Not total: 2 and 3 are noncomparable. Not equivalence: not symmetric.

Q9. A total order is a well order if:

C. A is just total order. C is the extra requirement. \((\mathbb{Z}^+,\le)\) is a well order; \((\mathbb{Z},\le)\) is not (no smallest integer).

Q10. Topological sorting of a finite poset \((S,R)\) produces:

C. Topological sorting iteratively picks minimal elements to produce a total order \(\preceq\) compatible with \(R\) (\(a\,R\,b\Rightarrow a\preceq b\)). The sort is not unique.
